   - castShadow = true, shadow.mapSize = 2048x2048
   - Add to camera (camera.add(this)), position (0, 0, 0)
   - Add to scene for transform updates
   - this.enabled = false, this.intensity = 7

2. Battery system:
   - battery = 100, maxBattery = 100
   - drainRate = 1.5 per second (when enabled)
   - drainRateSprint = 3.0 per second (when running)
   - recoveryRate = 0 (no recovery - battery is finite)

3. toggle() - F key:
   - Switches enabled state
   - If turning on: check battery > 0

4. update(delta, isSprinting, ghostDistance):
   - If enabled, drain battery * (isSprinting ? 2 : 1) * delta
   - If battery <= 0, disable flashlight

5. Ghost flicker:
   - flickerThreshold = 8 meters
   - If ghostDistance < threshold && enabled:
     - proximity = 1 - (ghostDistance / threshold)
     - Random flicker: if Math.random() < proximity * 0.3, intensity = random(1, 4)
     - Otherwise intensity = 7

6. Methods:
   - isEnabled(), getBattery(), getIntensity()

Reference Pattern 3 (shadow) and Pattern from RESEARCH.md (flicker).

  <files>src/player/Sanity.js</files>
  <action>
Create SanitySystem class:

1. Constructor():
   - sanity = 100, maxSanity = 100
   - decayRate = 2 per second (in darkness)
   - recoveryRate = 5 per second (in light, capped at 50% while flashlight off)
   - currentLightLevel = 0 (0-1 from light sources)

2. detectLightLevel(playerPosition, scene):
   - Cast ray upward, check distance to nearest light source
   - Check if in moonlight cone
   - Check if near candle (PointLight)
   - Calculate 0-1 based on nearest light distance
   - If flashlight on: lightLevel = 1 (immediate vicinity)

3. update(delta, lightLevel):
   - If lightLevel < 0.1: decay sanity (darkness)
   - If lightLevel >= 0.1: recover sanity (capped at 50% without flashlight)
   - If flashlight on: no decay, slight recovery possible
   - Clamp sanity between 0-100

4. getEffectIntensity():
   - Returns 0-1 based on sanity level
   - 100-50: 0 (no effect)
   - 50-25: gradual increase (0 to 0.5)
   - 25-10: sharp increase (0.5 to 0.8)
   - 10-0: max effect (0.8 to 1.0)

5. State getters:
   - getSanity(), isLow(), isCritical()
