<task type="auto">
  <name>Task 2: Collision detection system</name>
  <files>src/player/Collision.js</files>
  <action>
Create CollisionSystem class:

1. Constructor(scene):
   - Store reference to scene
   - Create Raycaster instance
   - collisionObjects array (walls, doors, furniture)
   - playerRadius = 0.5 (buffer from collisions)

2. registerObject(object):
   - Add mesh to collisionObjects array
   - Extract from door.js, furniture.js, walls

3. checkCollision(position, velocity):
   - Cast ray in movement direction
   - Use 4 directions: forward, back, left, right at 45° angles
   - If intersection < playerRadius, stop movement in that direction
   - Formula: velocity.addScaledVector(intersect.face.normal, -velocity.dot(intersect.face.normal))

4. handleCollision(controls, delta):
   - Get player position from controls.getObject().position
   - Check movement direction from velocity
   - Apply collision response

5. update(controls, velocity):
   - Call checkCollision each frame
   - Prevent player from entering collision objects

Use Pattern 4 from RESEARCH.md with multiple raycaster directions.
