Create AudioEngine (src/audio/AudioEngine.js):

1. Constructor():
   - this.ctx: AudioContext
   - this.masterGain: GainNode
   - this.compressor: DynamicsCompressorNode
   - this.listener: AudioListener
   - this.initialized: false

2. async init():
   - Create AudioContext (with webkit fallback)
   - Create DynamicsCompressor to prevent clipping
   - Create master Gain (volume control)
   - Connect: source -> masterGain -> compressor -> destination
   - Initialize SoundRegistry with noise buffers

3. async resume():
   - Check if ctx.state === 'suspended'
   - Call ctx.resume() if needed
   - Required for browser autoplay policy

4. updateListenerPosition(position, forward):
   - Sync AudioListener to player camera position/orientation
   - Called each frame in game loop
   - Position: x, y, z from camera.position
   - Forward: camera.getWorldDirection()

5. createSpatialSource(position):
   - Create PannerNode with HRTF model
   - Set distanceModel: 'inverse'
   - Set refDistance: 1, maxDistance: 30, rolloffFactor: 1.5
   - Return panner connected to masterGain

6. getOutput(): Return masterGain for non-spatial sounds

Create SoundRegistry (src/audio/SoundRegistry.js):

1. Constructor(audioCtx):
   - this.ctx: audioCtx
   - this.noiseBuffers: Map<string, AudioBuffer>

2. createNoiseBuffer(duration):
   - Create AudioBuffer with white noise
   - Fill with Math.random() * 2 - 1
   - Return buffer for reuse

3. createPinkNoiseBuffer(duration):
   - Create pink noise using Paul Kellet's algorithm
   - Better for ambient sounds than white noise

4. getNoiseBuffer(type): Return pre-generated buffer

Create Envelope (src/audio/synthesis/Envelope.js):

1. createAttackRelease(gainNode, attackTime, releaseTime, startTime):
   - Linear ramp from 0 to 1 over attackTime
   - Exponential ramp to 0.001 over releaseTime

2. createADSR(gainNode, a, d, s, r, startTime, duration):
   - Attack: 0 -> 1 over 'a' seconds
   - Decay: 1 -> 's' over 'd' seconds  
   - Sustain: hold 's' for duration
   - Release: 's' -> 0 over 'r' seconds

3. applyEnvelope(gainNode, params, now): Apply envelope to gain node

Create OscillatorSynth (src/audio/synthesis/OscillatorSynth.js):

1. createTone(audioCtx, freq, type, duration, volume):
   - Create OscillatorNode with specified type (sine, square, sawtooth, triangle)
   - Create GainNode with envelope
   - Apply frequency ramp for pitch bend effect
   - Connect: osc -> gain -> destination
   - Schedule start/stop with audioCtx.currentTime

2. createKick(audioCtx, freq, duration, volume):
   - Sine oscillator: freq starts at 150Hz, ramps to 55Hz
   - Quick attack (0.01s), quick decay (0.15s)
   - Lowpass filter at 500Hz for muffled sound
   - Base for footsteps

3. createPluck(audioCtx, freq, duration, volume):
   - Triangle oscillator with quick decay
   - Bandpass filter for plucked string sound
   - Base for music box

4. createSweep(audioCtx, startFreq, endFreq, duration, volume, type):
   - Frequency ramp from startFreq to endFreq
   - Used for door creaks, ghost screams

Create NoiseSynth (src/audio/synthesis/NoiseSynth.js):

1. createNoiseSource(audioCtx, buffer, loop):
   - Create BufferSource with noise buffer
   - Set loop flag
   - Return source node

2. createFilteredNoise(audioCtx, filterType, frequency, Q, volume):
   - Create noise source with buffer
   - Create BiquadFilterNode
   - Connect: source -> filter -> gain -> destination
   - Filter types: lowpass, bandpass, highpass

3. createWhisper(audioCtx, volume):
   - Bandpass filter at 800Hz, Q=2
   - LFO modulating filter frequency (3-5Hz, ±300Hz range)
   - Creates "vowel-like" formants

4. createStatic(audioCtx, volume):
   - Highpass filter at 3000Hz
   - Quick random gain modulation for crackle

   - Connect through spatial panner at position

5. playMusicBox(volume, position):
   - Triangle wave melody
   - Notes: C4, E4, G4, C5 pattern (523, 330, 392, 523 Hz)

   - Each note: quick attack, exponential decay
   - Slight detuning for realism
   - Connect through spatial panner

6. playSpiritVoice(text, volume):
