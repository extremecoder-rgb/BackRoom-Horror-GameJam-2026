Create PostProcessing class:

   - Pipeline order:

     d. Custom horror shader pass (vignette + chromatic + flicker)
     e. OutputPass (color space)

2. Custom horror effects shader (single pass):
   - Vertex: standard passthrough
   - Fragment uniforms:
     - vignetteOffset: 0.95, vignetteDarkness: 1.2
     - chromaticAmount: 0.003
     - flickerIntensity: 0.0
     - time: 0.0
   - Fragment:
     - Apply chromatic aberration (RGB offset based on distance from center)
     - Apply vignette (smoothstep based on distance)

3. Methods:
   - update(time): updates time uniform
   - render(): calls composer.render()
   - setSize(width, height): calls composer.setSize()
   - triggerFlicker(intensity=0.5, duration=0.5): sets flickerIntensity, fades via setTimeout
