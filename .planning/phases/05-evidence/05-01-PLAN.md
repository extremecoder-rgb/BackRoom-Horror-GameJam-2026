Create EMFReader (src/evidence/EMFReader.js):

1. Constructor():
   - currentLevel: 1-5 (1 = no activity, 5 = evidence)
   - maxRange: 3.5 meters
   - lastUpdate: timestamp for cooldown

2. checkEMF(ghost, playerPosition):
   - Calculate distance to ghost
   - If ghost.isInteracting && has EMF evidence: 25% chance level 5
   - If ghost.isManifested: level 4
   - Otherwise scale by distance
   - Return level

3. hasEvidence(): return currentLevel >= 5

Create Thermometer (src/evidence/Thermometer.js):

1. Constructor():
   - baseTemp: 20°C (room temp)
   - ghostRoomModifier: -20°C (freezing in ghost room)
   - currentTemp: reading

2. readTemperature(ghost, playerPosition, room):
   - If ghost in same room: return baseTemp + ghostRoomModifier
   - Otherwise: return baseTemp with slight variance
   - Store in currentTemp

3. hasEvidence(): return currentTemp <= 0

     - Add to detectedOrbs
   - Return orb count

4. renderOrbs(scene): draw orb particles in NV mode

1. Constructor():
   - isPlaced: false
   - hasWriting: false
   - writeDelay: 30000ms (30 seconds)
   - position: Vector3

2. place(position):
   - isPlaced = true
   - position = position
   - placedTime = now

3. checkWriting(ghost):
   - If !isPlaced: return false
   - If ghost in same room && has WRITING evidence:
     - If time since placed > writeDelay:
       - hasWriting = true
       - display message
   - Return hasWriting

4. pickup(): reset book state

   - If !isPresent: add to excludedEvidence, remove from confirmed

3. getPossibleGhosts(GhostTypes):
   - Filter GhostTypes where:
     - Has ALL confirmed evidence
     - Has NONE of excluded evidence
   - Return array of possible ghost types

   - evidenceCheckboxes: Map of evidence -> boolean
   - ghostFilter: new GhostFilter()
